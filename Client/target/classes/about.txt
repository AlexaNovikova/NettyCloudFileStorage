О программме.
Облачное хранилище MyCloud позволяет отправлять на сервер и получать обратно файлы.
Под заголовком "Мои файлы" размещен список файлов,расположенных на вашем компьютере в папке Files.
Непосредственно над списком указана текущая директория на локальном компьютере.
Под заголовком "Мои файлы" размещен список файлов,расположенных в папке (имя совпадает с вашим логином)
на облаке.
Непосредственно над списком указана текущая директория на облаке.
К имени файлов в списке добавлен их размер и формат.
К имени директории - [DIR].
Для перехода в новую директорию необходимо дважды щелкнуть по ней левой кнопкой мыши.

Для создания новой папки нажмите кнопку с изображением "плюс", расположенную в вернем правом углу списка файлов
и укажите наименование новой папки без пробелов(текст_текст_текст).

Для удаления или перемещения файлов выделите нужный файл и нажмите правую кнопку мыши. В
появившемся окне выберите нужную команду.
Delete - для удаления ( если вы выбрали папку, она будет удалена со всеми файлами, находящимися в ней).
Move - для перемещения в другую директорию. Для перемещения файла/папки укажите полный адрес новой папки
(начиная с  Client\src\Files для локальных файлов и с имени корневой папки(совпадает с логином) на облаке.

Для отправки файла на сервер выделите файл из списка "Мои файлы", щелчком левой кнопки мыши,
и нажмите кнопку Send( верхняя кнопка,расположенная между списками файлов).

Для получения файла с сервера на ваш компьютер - выделите файл в списке "Мое облако", щелчком левой кнопки мыши, и
 нажмите кнопку Get( нижняя кнопка,расположенная между списками файлов).

Для обновления списка файлов - нажмите кнопку Update - расположена посередине между списками файлов.
